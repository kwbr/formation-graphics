Formation Graphics Config Builder

Build a Match Config visually. Select a player, click the outfield positions they are comfortable playing, then copy or download the JSON.

Match inputs

These fields map directly to game_id, game_minutes, keepers, and kickoff starters.

Players

Choose exactly 7 kickoff starters. Both keepers must be starters.

Import JSON

Paste an existing config, or choose a JSON file. The page is standalone, so it does not upload anything.

Visual position choices

Select a player on the left. Click LB/RB/LM/CM/RM/ST to toggle comfortable positions. Position order is not meaningful.

Solver preview

Run the local FastAPI backend to preview real OR-Tools solver output. Start it with uv run formation-graphics-web.
